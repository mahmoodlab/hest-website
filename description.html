Dataset Description and Evaluation

Dataset format

For every sample, we provide: - The aligned Spatial Transcriptomics data in a standard ScanPy .h5ad file - A H&E WSI in pyramidal generic TIFF - CellViT nuclei segmentation - WSI patches - A JSON file containing metadatas for that sample An example showing the metadatas for a HEST sample is given below. A more detailed description of each sample is provided in the original paper.
